Résumé / description courte
JentOS is a web-based operating system that allows centralized management of internet access, secure storage of user files directly on the server, and provides a web interface with SSH terminal and file manager. Each user’s data is stored in their Linux home directory on the server, making it both an access control system and cloud-like storage solution.

Description détaillée

JentOS is a web-based OS designed to centralize internet access and file storage for a network of users.

Administrators can manage internet access by user, group, or site, using an admin interface powered by Django and iptables.

The project uses a custom DNS entry (jentos.mit) for user-friendly access.
